Simple Example React Native With Expo

feature : 
- login
- register
- logout

how to run : 
- clone this project
- npm install / yarn install
- expo start (make sure you have expo-cli on your device)
- scan barcode with expo client on your phone / emulator;


package used: 
- react & react native
- expo
- axios 
- @react-navigation 5.x.x
- redux
- redux persist
- asycn storage
